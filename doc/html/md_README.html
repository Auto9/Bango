Bango

KalOnline MMORPG Server Emulator written in C++ for Linux 64bit OS.

Quick installation

Install MySQL Server: sudo apt-get install mysql-server Install Libzdb: sudo apt-get install libzdb-dev Run \. build.sql in your database.
